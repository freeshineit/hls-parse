    HLS Tag 常量定义

    所有 M3U8 / HLS 标签名称的统一常量。 每个 tag 附带中英双语文档注释， 使用时通过 TAGS.TAG_NAME 引用，避免硬编码字符串。

    Example

    import { TAGS } from '@skax/hls-parse';
    // TAGS.EXT_X_TARGETDURATION === "#EXT-X-TARGETDURATION"
